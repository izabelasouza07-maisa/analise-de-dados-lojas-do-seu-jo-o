Análise de Vendas e Desempenho de Lojas - Projeto Data Science

Descrição


Este projeto tem como objetivo analisar dados de vendas de quatro lojas diferentes, permitindo extrair insights importantes sobre o desempenho comercial, satisfação dos clientes, custo operacional (frete) e mix de produtos vendidos. Com base nas análises quantitativas e visuais, o projeto orienta a decisão estratégica de qual loja pode ser mais vantajoso vender.

Conteúdo

Cálculo do faturamento total de cada loja.

Identificação das categorias de produtos mais e menos vendidas por loja.

Cálculo da média das avaliações dos clientes para cada loja.

Identificação dos produtos mais e menos vendidos em cada loja.

Análise do custo médio do frete pago pelos clientes.

Visualizações gráficas para ilustrar os principais insights.

Relatório final com recomendação de qual loja vender, fundamentado nos dados analisados.

Tecnologias Utilizadas
Python 3.x

Pandas para manipulação e análise dos dados

Matplotlib para visualização gráfica

Jupyter Notebook (opcional) para execução interativa

Resultados esperados
